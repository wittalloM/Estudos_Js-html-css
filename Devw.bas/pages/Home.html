Um breve disclaimer sobre o JS.

O javascript é uma das linguagens de programação mais utilizadas do mundo, no mercado de navegadores ela é amplamente adotada e sofre constantes aperfeiçoamentos, agora veremos um pouco sobre a história do Js bem como as suas tranformações ao longo de sua história, seus propósitos e usos...

História.

Criada em 1995 pelo engenheiro Brendan Eich a pedido da NetScape Communication, a linguagem de script, lançada em setembro do mesmo ano sobe o nome de LiveScript. JavaScript (frequentemente abreviado como JS) é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Sua primeira implementação se deu no NetScape navigator 2.0, arquitetada para adicionar dinamicidade e iteratividade as páginas web, ela foi amplamente adotada pela recém iniciada comunidade de desenvolvedores web por suas características. Após vencer uma árdua contenda com a gigante microsoft pela adoção em massa de tecnologias de script pra web com o Javascript, o Netscape Navigator conquistou a todos, e logo com a padronização ECMA em 97, se tornou então indispensável para criar experiências dinâmicas e fluídas na web moderna.

JS na atualidade.

Hoje em dia o Javascript se tornou uma das principais ferramentas web, juntamente com o HTML e o CSS. No âmbito da área de desenvolvimento de sofware, a linguagem pode e já têm sido usada para resolver problemas nas mais diversas áreas, tais como Robótica, Jogos, Manipulação de arquivos de texto(PDF) e etc... Uma das tecnologias derivadas do JS que se destaca bastante é o ambiente de execução JavaScript NodeJS... está ferramenta nos possibilita ter acesso a diversas funcionalidades que seriam díficeis ou impossíveis de serem implementadas em um ambiente de navegador; Um exemplo é o acesso a arquivos. Um código JavaScript no navegador não consegue acessar os arquivos do usuário, pois seria um problema de segurança você entrar em um site e ele vasculhar seu computador. Já que o Node.js está instalado em sua máquina, o JavaScript tem liberdade de acessar os arquivos contidos no HD. Não podemos nos esquecer ainda do NPM o gerenciador de pacotes embarcado no NodeJS.
